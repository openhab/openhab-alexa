/* One Sentence Description */

Contrôlez votre système openHAB 2 avec Alexa et myopenHAB.org

/* Description détaillée */

La skill maison connectée openHAB vous permet de contrôler les lumières, les thermostats et les autres éléments connectés sur un système openHAB 2 via le service de cloud myopenHAB.org. 

Le projet ouvert Home Automation Bus (openHAB) vise à fournir une plate-forme d’intégration universelle pour toutes les choses autour de domotique. Il est conçu pour être absolument neutre et indépendant du matériel/protocole. openHAB regroupe des systèmes de bus différents, les périphériques matériels et protocoles d’interface. Technologies prises en charge incluent KNX, Modbus, DMX, Philips Hue, PLCBus, Sonos, Plugwise, IHC/Elko, RFXCOM, Homematic, Koubachi, Pulseaudio, Samsung TV, Bluetooth, One-Wire, Novelan, astérisque, équipements de série génériques et bien d’autres. Vous trouverez openHAB sur http://www.openhab.org/

GUIDE DE DÉMARRAGE

Assurez-vous d’abord que vous avez votre système openHAB configuré et connecté à l’openHAB Cloud service.  openHAB Cloud comptes sont gratuits, visitez https://myopenhab.org pour plus d’informations. 

Dans la configuration de votre openHAB, ajouter un tag vocale pour les éléments que vous souhaitez exposer à Alexa. Les éléments de type « Number »,  « Dimmer »,  « Switch » ou  « Thermostat » (de type string ou number), sont supportés. 

Pour terminer, activer la skill openHAB pour Alexa.  À l'invite, saisir vos identifiants de openHAB Cloud pour associer votre système à Alexa. 

Une fois associé, les « items » configurés avec des tags vocaux seront découverts automatiquement et disponibles pour Alexa.  IF you add additional items after linking, just ask Alexa to "Discover Devices" to add them to your Alexa account.

You can control items in your openHAB system directly, for example:
 - Alexa, turn on Kitchen

You can also add your items to Alexa groups to control multiple items at once, for example:
 -Alexa, set Bedroom to 50 percent

Scenes and Routines can also be used to control multiple openHAB items, for example:
 - Alexa, turn on Bed Time.
